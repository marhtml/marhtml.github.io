# CV Web - Martina Trinidad Movia

Este proyecto es un CV técnico desarrollado en HTML y CSS puro, sin frameworks ni JavaScript.

## Descripción

Página web que presenta mi perfil profesional como desarrolladora web fullstack orientada a JavaScript. Incluye mis tecnologías, proyectos académicos con links a los repositorios y datos de contacto.

## Tecnologías utilizadas

- HTML semántico (header, main, section, article, footer)
- CSS (variables, flexbox, diseño responsivo)
- Google Fonts (Syne + Inter)

## Funcionalidades

- Presentación de perfil profesional
- Lista de tecnologías del stack
- Sección de proyectos con enlaces a GitHub y demos
- Información de contacto
